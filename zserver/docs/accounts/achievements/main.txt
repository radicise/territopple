this file documents account achievements / badges

Achievement:
this data is stored within an account
{
    id<Int32>: id of this achievement,
    data<any>: extra achievement data
}


Achievement Information:
this data is stored in a seperate location and describes an achievement
{
    id<Int32>: achievement id,
    name<string>: internal achievement name,
    granting<{
        prereqs<Prerequisites>: prerequisite achievements,
        acts<ActionId>: id of actions to complete to be awarded this achievement see "./action.txt",
        init<any>: initial data
    }>: information on what is required to achieve this achievement,
    evo<Mutator[]>: how to mutate this data,
    display<Display>: how to display the achievement see "./display.txt",
    population<Int64>: how many accounts have this achievement
}

Prerequisite {
    id<Int32>: prerequisite achievement id,
    cond<Buffer<AchievementCode + D2: Readonly>>: condition
}
Prerequisites {
    comb<Int32>: 0:all,1:any,
    sub<Prerequisite[]>,
}

Mutator {
    act<ActionId>: id of Actions to trigger this mutator,
    dis<Int32>: discriminator,
    mut<Buffer<AchievementCode>>: how to mutate,
}
